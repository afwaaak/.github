## What We Build

WAAAK creates practical tools and real-time systems for installation sites, projection environments, live operation, and interactive content.

- Projection calibration and masking tools
- Kiosk and event operation software
- Unreal Engine and real-time media workflows
- Spatial video and preview utilities
- Field-ready systems designed for repeated daily use

## Featured Project

### [Projection Calibrator](https://github.com/afwaaak/projection-calibrator)

A fullscreen Electron player for portrait projection playback, transform adjustment, and edge mask calibration.

- Loads video and calibration from `config.json`
- Supports fullscreen field operation
- Saves transform, scale, rotation, skew, and mask settings
- Packs into a Windows executable for on-site deployment
